The primary artifact presented in this ePortfolio is a Guitar Scale Viewer application developed using React and modern JavaScript. I selected this artifact because it demonstrates my ability to design and implement a functional, interactive software system while applying core computer science concepts such as data structures, algorithms, and modular design.

Originally, the application allowed users to visualize musical scales across a guitar fretboard. For this capstone enhancement, I focused on improving code organization, readability, and maintainability, as well as clarifying the separation of concerns between data generation, user interaction, and visual rendering. These enhancements were made to better align the project with industry best practices and software engineering principles.

This artifact showcases my skills in front-end development, algorithmic problem solving, and iterative refinement based on evaluation and reflection. The enhancements demonstrate my growth in applying structured design patterns, reasoning about performance, and presenting technical work in a clear and professional manner suitable for real-world software development.
